# Learn Flutters

目录结构

```
lib
├── common                // Utils, Helpers..
├── data                  // [domain layer]
│   ├── mappers           // data mapper
│   ├── models            // models
│   ├── repositories
│   └── usecases
├── main.dart
└── ui
    ├── pages             // page level component
    └── widgets           // basic widgets
```
